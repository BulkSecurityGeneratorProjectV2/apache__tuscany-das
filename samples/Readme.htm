DAS Samples

The DAS Java Samples demonstrates the DAS features and how to get started with them. Below a list of available samples:

Companyweb

This stand-alone sample demonstrates DAS in the context of a simple web application.

Customer

This stand-alone sample demonstrates DAS in the context of a simple java application.

Ajax DAS Web

This web sample demonstrates DAS advanced features in the context of a simple web application using Ajax. Some of the features demonstrated are : Basic CRUD, Optimistic Concurrently Control (OCC), Result Set Shape, Column Converter,Integration with dbConfig
